Your Personal Website Bookmarking Tool

BookMarker

Project Overview

"Bookmarker" is a user-friendly web application that simplifies website bookmarking, creating a seamless experience using HTML5, CSS3, Bootstrap 5, and JavaScript. This project is designed to leverage CRUD operations, allowing users to effortlessly add, visit, and manage their bookmarked websites.

    Key Features:
  • Create: Easily add websites to your bookmark list by providing the site's name and URL through a user-friendly input field.
  • Read (Visit): Navigate to bookmarked websites directly from the application, providing a quick and convenient way to access your favorite online destinations.
  • Update (Prevent Duplication): Bookmarker prevents the duplication of website names, ensuring a clean and organized bookmark list.
  • Delete: Seamlessly manage your bookmarks by removing websites that are no longer needed, enhancing the overall user experience.
  • Validation (Regex): Implement regex validation to ensure data integrity, creating a robust and error-resistant bookmarking tool.

    Technologies Used:
  • HTML5, CSS3, Bootstrap 5: Craft a visually appealing and responsive design, providing an optimal user experience across devices.
  • JavaScript: Enable dynamic and interactive elements, facilitating CRUD operations for seamless website management.
